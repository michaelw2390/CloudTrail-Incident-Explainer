CloudTrail Incident Explainer
A web app that analyzes AWS CloudTrail logs to detect suspicious activity and reconstruct an attack timeline.

Features
- Detects risky cloud actions
- Builds an attack timeline
- Provides security insights

How to Use
1. Paste CloudTrail JSON
2. Click "Analyze Incident"
3. Review results

- HTML
