# CloudTrail Incident Explainer

A lightweight web app that analyzes AWS CloudTrail logs to identify suspicious activity, reconstruct an attack timeline, and provide security insights.

## Features
- Parses CloudTrail JSON input
- Detects high-risk events (e.g., security group exposure, access key creation)
- Builds a chronological attack timeline
- Provides recommended remediation steps

## Technologies
- HTML
- JavaScript

## Purpose
This project demonstrates practical SOC analyst skills including log analysis, threat detection, and incident investigation.
